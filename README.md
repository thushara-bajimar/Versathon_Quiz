# YOUR QUIZ PLATFORM
A Simple Console-Based Quiz Application developed using C++.
This project demonstrates basic programming concepts like File Handling, functions, sorting etc.

## FEATURES
- Start Quiz
- MCQ
- Calculated Score
- Badges
- Sorted Leaderboard
- Exit option
- File-based data storage

## CONCEPTS USED
- C++
- File Handling
- Vectors
- Bubble Sort
- Functions
- Loops & Conditions

## FILES USED
- question.txt
- score.txt
- main.cpp
